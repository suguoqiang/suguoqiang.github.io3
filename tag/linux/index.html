    linux-安装MongoDB、RabbitMQ、Git

    # 安装MongoDB

    $ curl -O https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-3.0.6.tgz    # 下载
$ tar -zxvf mongodb-linux-x86_64-3.0.6.tgz                                   # 解压

$ mv  mongodb-linux-x86_64-3.0.6/ /usr/local/mongodb                         # 将解压包拷贝到指定目录
$ export PATH=<mongodb-install-directory>/bin:$PATH    #添加path
$ mkdir -p /data/db    #创建数据库
$ ./mongod    #运行服务

    # 安装RabbitMQ

    1、安装：

    #安装epel源[EPEL (Extra Packages for Enterprise Linux，企业版Linux的额外软件包) 是Fedora小组维护的一个软件仓库项目为RHEL/CentOS提供他们默认不提供的软件.]
$ rpm -Uvh https://dl.fedoraproject.org/pub/epel/epel-release-latest-6.noarch.rpm
# 扩展erlang源安装最新的erlang
$ rpm -Uvh http://packages.erlang-solutions.com/erlang-solutions-1.0-1.noarch.rpm
$ yum -y install erlang
# 安装RabbitMQ
$ wget http://www.rabbitmq.com/releases/rabbitmq-server/v3.6.1/rabbitmq-server-3.6.1-1.noarch.rpm
$ rpm --import https://www.rabbitmq.com/rabbitmq-signing-key-public.asc
$ yum install rabbitmq-server-3.6.1-1.noarch.rpm

    linux-安装Jenkins，部署SpringBoot工程

    # 安装准备

    安装前确认已安装了JDK、Git、Maven,并配置相应的环境变量。

    1、安装maven：

    wget http://mirror.bit.edu.cn/apache/maven/maven-3/3.5.4/binaries/apache-maven-3.5.4-bin.tar.gz
## 解压
tar vxf apache-maven-3.5.0-bin.tar.gz
## 移动
mv apache-maven-3.5.0 /usr/local/maven3

    2、修改环境变量，在/etc/profile中添加：

    MAVEN_HOME=/usr/local/maven3
export MAVEN_HOME
export PATH=${PATH}:${MAVEN_HOME}/bin

    3、执行source /etc/profile使环境变量生效。然后mvn -v查看是否安装成功。

    linux运维--详细命令大全

    # linux运维

    # cd 命令

    命令格式 cd [目录名] 切换当前目录到指定目录

    • cd /进入系统根目录
    • cd .. 退回上一级目录（进入当前目录的父目录）
    • cd 或者 cd~ 使用cd命令进入当前用户主目录
    • cd /opt/soft 跳转到指定目录（跳转到指定目录，从根目录开始，目录名称前加 / ,当前目录内的子目录直接写名称即可）
    • cd - 返回进入此目录前所在的目录
    • cd !$ 把上个命令的参数作为cd参数使用

    查看是否安装完成：

    $ node -v

    linux学习知识点梳理

    服务器系统：centOs 7

    # 防火墙相关

    注意：Centos 7使用firewalld代替了原来的iptables。所以7以前版本的iptables是不能用的。

    # 开启防火墙

    $ systemctl start firewalld.service

    一般链接到服务器正常，但是无法访问服务，首先想到的就是防火墙是否正常开启。

    # 关闭防火墙

    $ systemctl stop firewalld.service             #停止firewall
$ systemctl disable firewalld.service        #禁止firewall开机启动
$ systemctl enable firewalld.service          #firewall开机启动

    # 开放端口

    $ firewall-cmd --add-port=80/tcp --permanent   #永久添加80端口
$ firewall-cmd --reload                          #重新载入配置

## zone -- 作用域
## add-port=80/tcp -- 添加端口，格式为：端口/通讯协议
## permanent -- 永久生效，没有此参数重启后失效
$ firewall-cmd --zone=public --add-port=3306/tcp --permanent

    # 查看端口占用情况

    $ lsof -i:80

    # Ngix安装

    此处的安装为源码安装，yum安装方式参见 nginx标签 栏目，至于区别自行百度。推荐源码安装方式，可以安装最新版本

    1、安装编译工具及库文件、依赖包

    $ yum install gcc-c++
$ yum -y install zlib zlib-devel
$ yum install -y openssl openssl-devel

    2、安装PCRE（作用：使Nginx支持rewrite功能）

    $ yum -y install pcre pcre-devel

    以上两个步骤可以合并：yum -y install gcc pcre-devel zlib-devel openssl openssl-devel

    3、下载安装nginx,或者去官网下载所需版本

    wget http://nginx.org/download/nginx-1.19.2.tar.gz

    linux环境搭建之安装JDK和Mysql

    服务器系统：centOs 7

    注意版本号要与服务器对应

    安装前准备：已下载好对应的JDK和Mysql安装包 当然，你也可以通过wget +地址的方式进行在线下载

    # wget工具安装

    $ yum -y install wget

    在虚拟机中可能会出现安装失败，解决办法： 进入编辑模式vi /etc/sysconfig/network-scripts/ifcfg-enp0s3 ， 将ONBOOT=no改为ONBOOT=yes，就OK。 保存后重启网卡： service network restart即可
